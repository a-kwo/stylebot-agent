│                                                                                                                      │
│ Context                                                                                                              │
│                                                                                                                      │
│ StyleBot currently drops users directly into a login/register screen at /. There's no marketing presence — no way    │
│ for a visitor to understand what StyleBot is before committing to sign up. We need a polished, engaging landing page │
│  at / that sells the product, with the auth page moved to /login. Existing app pages must not be functionally        │
│ modified.                                                                                                            │
│                                                                                                                      │

│                                                                                                                      │

│                                                                                                                      │
│ New: / = landing page (public, no auth guard). /login = auth page (moved).                                           │

│ Design Guidelines                                                                                                    │
│                                                                                                                      │
│ - Reuse existing design system entirely — no new colors, fonts, or CSS classes                                       │
│ - Colors: cream bg, accent (#7c2d3e) CTAs, gold accents, ink text                                                    │
│ - Dark mode: all components use dark: variants with surface-dark-*                                                   │
│ - Typography: font-display (Cormorant Garamond) for headlines, font-sans (DM Sans) for body                          │
│ - CSS classes: btn-primary, btn-secondary, card, card-hover, input-field                                             │
│ - Animations: Framer Motion with existing easing curve [0.22, 1, 0.36, 1]                                            │
│ - noise texture inherited from body class in layout.tsx                                                              │
│ - Tone: Premium & editorial — elegant, fashion-magazine feel. Refined aspirational language. Serif display           │
│ headlines, considered whitespace, understated confidence. Think Vogue meets modern tech.                             │
